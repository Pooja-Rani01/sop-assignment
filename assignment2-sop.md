# Standard Operating Procedure (SOP)
## Virtual Machine Creation using VMware vSphere

**Company Name:** MITT Lab Environment  
**Location:** Winnipeg, Canada  
**Version:** 1.0  
**Date:** March 30, 2026  

---

## 1. Purpose
The purpose of this SOP is to provide a standardized process for creating a Virtual Machine (VM) using VMware vSphere.

---

## 2. Scope
This SOP applies to IT staff and students responsible for VM creation.

---

## 3. Procedure Steps

### Step 1: Planning
- Identify purpose  
- Decide CPU, RAM, storage  

### Step 2: Configuration
- Open VMware  
- Create new VM  
- Allocate resources  

### Step 3: Install OS
- Attach ISO  
- Install OS  

### Step 4: Testing
- Check VM  
- Test network  

### Step 5: Documentation
- Save VM details  

---

## 4. Risks
- Resource issues  
- Network errors
